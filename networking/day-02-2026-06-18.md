# Networking & OS Fundamentals - Day 02

## IPv6

### What is IPv6?

IPv6 is the successor to IPv4 and was created to solve IPv4 address exhaustion.

Example:

```text
2001:0db8:85a3:0000:0000:8a2e:0370:7334
```

### Benefits

- Larger address space
- Improved routing
- Better scalability
- Built-in support for modern networking

---

## MAC Address

### What is a MAC Address?

A Media Access Control (MAC) address is a unique identifier assigned to a network interface card (NIC).

Example:

```text
00:1A:2B:3C:4D:5E
```

### Characteristics

- Layer 2 address
- Used inside local networks
- Assigned to network hardware

---

## MAC Spoofing

### What is MAC Spoofing?

Changing a device's MAC address to appear as another device.

### Using macchanger

```bash
macchanger -r eth0
```

### Manual Method

```bash
ip link set eth0 down
ip link set eth0 address XX:XX:XX:XX:XX:XX
ip link set eth0 up
```

### Why Learn It?

- Privacy testing
- Security research
- Network analysis

## Network Ports

### Why Do Ports Exist?

Ports allow multiple services to communicate using the same IP address.

Example:

```text
IP Address = House Address
Port = Room Number
```

### Port Ranges

#### Well-Known Ports

```text
0 - 1023
```

Examples:

```text
20  FTP
22  SSH
25  SMTP
53  DNS
80  HTTP
443 HTTPS
```

#### Registered Ports

```text
1024 - 49151
```

#### Dynamic / Private Ports

```text
49152 - 65535
```

## DNS (Domain Name System)

### Why DNS Exists

Humans remember names more easily than IP addresses.

Instead of:

```text
142.250.193.78
```

we use:

```text
google.com
```

### Basic Process

1. User enters domain name.
2. DNS resolver receives request.
3. DNS server returns IP address.
4. Browser connects to destination server.

## Servers

### What is a Server?

A server is a system that provides services, data or resources to clients.

Examples:

- Web Server
- DNS Server
- Mail Server
- File Server

---

## Client-Server Model

### Client

Requests a service.

Examples:

- Browser
- Mobile App

### Server

Provides a service.

Examples:

- Website
- API
- Database

Example Flow:

```text
Browser
   ↓
DNS
   ↓
Web Server
   ↓
Response Returned
```
