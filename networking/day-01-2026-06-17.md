# Networking & OS Fundamentals - Day 01

## Operating System Basics

### What is an Operating System?

An Operating System (OS) is software that acts as an interface between users and computer hardware.

Examples:
- Linux
- Windows
- macOS
- Android

### Functions of an Operating System

- Process Management
- Memory Management
- File Management
- Device Management
- User Management
- Security and Permissions

## Networking Fundamentals

### What is an IP Address?

An IP Address is a unique identifier assigned to devices on a network.

Example:

```text
192.168.1.10
```

### IPv4 Structure

IPv4 consists of 32 bits divided into 4 octets.

Example:

```text
192.168.1.10
```

---

## IP Classes

### Class A

```text
1.0.0.0 - 126.255.255.255
```

Default Mask:

```text
255.0.0.0
```

### Class B

```text
128.0.0.0 - 191.255.255.255
```

Default Mask:

```text
255.255.0.0
```

### Class C

```text
192.0.0.0 - 223.255.255.255
```

Default Mask:

```text
255.255.255.0
```

---

## Classful vs Classless Addressing

### Classful

Uses predefined classes (A, B, C).

### Classless (CIDR)

Uses flexible subnet masks.

Example:

```text
192.168.1.0/24
192.168.1.0/26
```

---

## Subnet Masks

Examples:

```text
255.255.255.0
255.255.255.192
255.255.255.224
```

Purpose:

- Divide networks into smaller subnetworks.
- Improve network organization.

---

## CIDR Notation

Examples:

```text
/24
/25
/26
/27
/28
```

CIDR specifies how many bits belong to the network portion.

Example:

```text
192.168.1.0/24
```

---

## Basic Subnetting

Learned how to determine:

### Network ID

Identifies the network.

Example:

```text
192.168.1.0
```

### Broadcast Address

Used to communicate with all hosts in a subnet.

Example:

```text
192.168.1.255
```

### Host Range

Example:

```text
192.168.1.1 - 192.168.1.254
```

---

## Ping

### What is Ping?

Ping is used to test connectivity between devices.

Example:

```bash
ping google.com
```

Purpose:

- Check connectivity
- Measure latency
- Troubleshoot networks
