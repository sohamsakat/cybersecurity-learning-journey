# Linux Day 02

## Commands Practiced

### ssh
Connect to a remote system securely.

```bash
ssh
```

### pwd
Display the current working directory.

```bash
pwd
```

### ls
List files and directories.

```bash
ls
```

### ls -la
List all files, including hidden files, with detailed information.

```bash
ls -la
```

### cd
Change directory.

```bash
cd
```

### cat
Display file contents.

```bash
cat filename
```

### file
Identify a file's type.

```bash
file filename
```

### find
Search for files and directories.

```bash
find
```

---

## Concepts Learned

### Hidden Files
Files beginning with `.` are hidden by default.

Example:

```bash
.hiddenfile
```

### Spaces in Filenames
Filenames containing spaces must be quoted or escaped.

Example:

```bash
cat "my file.txt"
```

### Special Filename '-'
Some Linux utilities treat `-` as standard input/output, requiring special handling.

### File Types
Used the `file` command to identify different file types.

### Finding Files by Size
Learned to search for files using size-related filters.
